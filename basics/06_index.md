## INDEX USED IN STRING

### Print letters of a string in fwd/reverse order using index
```py
name = "PANDA"
print(name[0])
print(name[1])
print(name[2])
print(name[3])
print(name[4])
print()
print(name[-1])
print(name[-2])
print(name[-3])
print(name[-4])
print(name[-5])
```
