## LOOPS

### For loop (Print table of 5)
```py
for i in range (1, 11):
    print ("5 x", i, "=", (i)*5)
```

### For loop (Print any multiplication table)
```py
n = int(input("Enter a number for multiplication table: "))

for i in range (1, 11):
    print (n, "x", i, "=", (i)*(n))
```
