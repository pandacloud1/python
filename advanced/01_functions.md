## FUNCTIONS
- Functions help you to write your own code
- You need to define the function & call it to execute

### Average of 3 numbers
```py
def average(a,b,c):
    avg = (a+b+c)/3
    print(avg)

average(1,2,3)
average(4,5,6)
average(7,8,9)
```

### Using return to store value in variable
```py
def average(a,b,c):
    avg = (a+b+c)/3
    return avg

a1 = average(1,2,3)
a2 = average(4,5,6)
a3 = average(7,8,9)
print(a1,a2,a3)
```
